# Aplicativo de Tarefas

Aplicativo simples de tarefas desenvolvido em *HTML, CSS e JavaScript*.

O usuário pode:
- Adicionar tarefas  
- Marcar como concluídas  
- Ativar modo escuro  
- Escolher categoria, data e horário  
- Exportar e importar tarefas em JSON  

Todos os dados são salvos localmente usando *localStorage*.
